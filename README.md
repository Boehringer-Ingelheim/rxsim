# rxsim

[![R-CMD-check](https://github.com/saumil-sh/rxsim/actions/workflows/R-CMD-check.yaml/badge.svg)](https://github.com/saumil-sh/rxsim/actions/workflows/R-CMD-check.yaml)

The goal of rxsim is to reduce friction in simulating randomized
clinical trials (RCT).

## Installation

You can install the development version of rxsim from
[GitHub](https://github.com/Boehringer-Ingelheim/rxsim) with:

``` r
install.packages("pak")
pak::pak("Boehringer-Ingelheim/rxsim")
```

## Usage

Checkout the vignette that shows a basic example:

``` r
vignette("example-usecase", package="rxsim")
```
