# Convert Vector to Population Data Frame

Formats a numeric vector as a population data frame.

## Usage

``` r
vector_to_dataframe(data)
```

## Arguments

- data:

  `numeric` vector of population values.

## Value

`data.frame` with columns: `id`, `data`, `readout_time`.
