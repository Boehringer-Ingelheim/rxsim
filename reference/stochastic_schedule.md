# Generate a Stochastic Enrollment and Dropout Schedule

Creates a time-indexed schedule of enrollment and dropout events across
arms by sampling inter-event times from user-supplied distribution
functions. Each call produces a different realization, capturing natural
variability in study timelines.

## Usage

``` r
stochastic_schedule(sample_size, arms, allocation, enrollment, dropout)
```

## Arguments

- sample_size:

  `integer` Trial sample size.

- arms:

  `character` vector of arm identifiers.

- allocation:

  `numeric` vector of allocation ratios.

- enrollment:

  `function` that takes `n` and returns `n` inter-arrival times (e.g.
  `function(n) rexp(n, rate = 1)`).

- dropout:

  `function` that takes `n` and returns `n` inter-dropout times.

## Value

`data.frame` with columns: `time`, `arm`, `enroll` (always 1), `drop`
(always 0 or 1). One row per subject event, sorted by `time`.

## Details

Use this when trial-duration variability is substantively important. For
a fixed, reproducible schedule see
[`deterministic_schedule()`](https://boehringer-ingelheim.github.io/rxsim/reference/deterministic_schedule.md).

## See also

[`deterministic_schedule()`](https://boehringer-ingelheim.github.io/rxsim/reference/deterministic_schedule.md)
for piecewise-constant rates,
[`add_timepoints()`](https://boehringer-ingelheim.github.io/rxsim/reference/add_timepoints.md)
to attach to a `Timer`.
